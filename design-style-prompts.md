## 1. Terminal / AI Command Center (New!)

Create a webpage in a futuristic, minimal, terminal‑inspired UI style that resembles an AI command center. The design should feel like a Next.js agentic workflow console with a dark, high‑contrast aesthetic. Apply the following style rules:

1. **Visual Aesthetic**
   - Dark background (#000000 to #0A0A0A)
   - High‑contrast text in white, neon green, or cyan
   - Occasional accent colors: electric blue, magenta, amber
   - Use monospace or semi‑monospace fonts (e.g., JetBrains Mono, IBM Plex Mono, Space Mono)
   - Minimal shadows, no gradients, no rounded corners unless subtle
   - Everything should feel like a HUD, terminal, or system log

2. **Layout Rules**
   - Centered content with generous spacing
   - Large, single input field styled like a command prompt
   - Section headers styled like system messages (e.g., [DISPATCH AGENTS])
   - Use horizontal dividers that look like terminal separators (────────────)
   - Avoid traditional UI chrome (no navbars, no cards, no panels)
   - Use modular blocks that appear like logs or agent outputs

3. **Interaction Model**
   - The page should feel alive, even if static
   - Include system‑status language such as: IDLE — Waiting for signals… Neural Stream Active Swarm Ready Dispatching Agents…
   - Use micro‑animations or implied motion (e.g., blinking cursor, pulsing dots)
   - Inputs should feel like commands, not forms

4. **Tone & Copywriting**
   - Short, clipped, system‑style phrases
   - Avoid marketing language
   - Use verbs like deploy, dispatch, stream, orchestrate, initialize
   - Everything should feel like an AI operations dashboard

5. **Components to Include**
   - Hero section with a system‑console title (e.g., Orchestrated AI Workflow v2.0)
   - Command input with placeholder like: Enter a topic needed for viral content…
   - Status readouts such as: Neural Stream: ACTIVE Agent Swarm: READY
   - Output modules styled like terminal logs
   - Footer with minimal system metadata (e.g., Build 2.0.14 — All systems nominal)

6. **Optional Enhancements**
   - Subtle scanline effect
   - Blinking caret animation
   - ASCII‑style dividers
   - Agent "cards" that look like log entries, not UI components

## 2. Swiss / International Style

Create a webpage in the Swiss / International Style. This is a classic design movement characterized by:

- **Visual Aesthetic**: Clean, objective, mathematical precision
- **Colors**: Primary accent (red #FF3B30), black, white, light grays
- **Typography**: Helvetica Now Display for headings, Inter for body text
- **Layout**: Strict grid systems, asymmetric layouts, generous negative space
- **Elements**: Sans-serif typography everywhere, objective photography, strong visual hierarchy
- **Feel**: Professional, objective, timeless, trustworthy
- **Details**: Flush-left text, rag right, mathematical alignment, clear visual hierarchy
